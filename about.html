About...

...Daky in Monsterland:

Daky in Monsterland is a re-imagining of a game that first appeared in the early 80's called Robots. Robots was also released in the mid-80's for the original Mac with a Doctor Who theme under the name "Daleks." In this re-branding of the concept, I've taken a hapless but well-meaning dragon named Daky- a character invented by my wife - and given the game a more modern, more colorful look.

Daky in Monsterland was created using HTML5, CSS3, javascript, and jQuery, alongside an HTML framework designed for making games called Phaser, and a lightweight CSS framework called Skeleton. The art assests were provided by my wife, Becky Chapin.

...the Developer:
